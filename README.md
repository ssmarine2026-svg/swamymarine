# Swamy Marine Services Website

Static one-page corporate website for Swamy Marine Services (OPC) Private Limited.

## GitHub Pages

1. Push these files to the GitHub repository.
2. Open repository Settings.
3. Go to Pages.
4. Select Deploy from a branch.
5. Choose the `main` branch and `/root`.
6. Save.

The contact form uses FormSubmit and may require first-time verification from `support@swamymarine.co.in`.
